Proposta do Projeto

O Dice Dragons é uma plataforma online voltada para os jogadores de D&D que desejam rolar seus dados de forma digital.

Vantagens

A plataforma permite os jogadores selecionarem o tipo de dado, de rolagem (vantagem, desvantagem e soma), adicionar bonus. Além disso, você criar seu personagem e adicionar rolagens fixas como seu ataque, dano, habilidades ou magias para facilitar e agilizar sua jogatina de RPG.

Dashboard

Além de conseguir ver os resultados das rolagens você pode ver seu último resultado abaixo dos campos para rolar o dado. O site também proporciona uma dashboard que permite o usuário verificar suas últimas rolagens no d20, quantos acertos e erros críticos, quantas rolagens o usuário realizou e qual dado ele mais rolou.

Sobre mim

Eu tenho muito carinho por RPG, por isso fiz esse projeto para jogadores de de D&D. RPG me ajudou a conquistar novas amizades e a vencer minha timidez. Além disso, fiz esse projeto para desenvolvimento tecnico.
